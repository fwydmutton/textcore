# textcore

A simple online notepad that lets you share text using a link. No login. No backend. Fully client-side.

---
# Use the site here


---
## Features

* Paste any text
* Generate shareable link
* Works offline
* No data stored on server
* Fully anonymous

## How it works

* Text is compressed using lz-string
* Stored in the URL hash
* Opening the link restores the text

## Limitations

* URL length limits apply
* Not suitable for very large files

## Usage

* Open the site
* Paste text
* Click Generate Link
* Share the link

## Privacy

* No tracking
* No login
* Data never leaves your browser
